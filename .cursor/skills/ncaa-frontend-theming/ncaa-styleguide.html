# NCAA Commissioner App — Frontend Rules

## Theme
Dark retro aesthetic. 1970s gridiron. Sharp corners, monospace type, teal + gold palette.

## Fonts
- Display (stats, headings, names): "Bebas Neue" — always letter-spacing ≥ 2px, uppercase
- Body/UI (everything else): "IBM Plex Mono" — 10–14px, uppercase labels with letter-spacing

## Colors (always use CSS variables, never hardcode)
--color-bg-base: #001219        Page root background
--color-bg-surface: #000e12     Sidebar, topbar, inputs
--color-bg-raised: #001a22      Cards, panels
--color-teal-700: #005f73       Borders, dividers
--color-teal-500: #0a9396       Active state, icons, primary select
--color-teal-300: #94d2bd       Body text, labels
--color-gold-500: #ee9b00       Primary CTA, active nav, highlights
--color-gold-700: #ca6702       Secondary amber accent
--color-gold-900: #bb3e03       Rust, hover on gold
--color-danger-500: #9b2226     Destructive border
--color-danger-text: #e87b7f    Destructive text
--color-text-primary: #e9d8a6  Main body text (cream)
--color-text-muted: #94d2bd    Secondary text

## Components
- .btn — base button: padding 7px 16px, font 10px, letter-spacing 1.5px, uppercase, border-radius 0
  Variants: .btn-primary .btn-secondary .btn-ghost .btn-gold .btn-danger
- .panel — content card: bg-raised, border teal-700, border-top 2px accent color
  Variants: .panel--gold .panel--amber .panel--rust .panel--neutral
- .field-label — 8px, letter-spacing 2px, teal-500, uppercase, margin-bottom 4px
- select / input — bg-surface, border teal-700, padding 7px 12px, border-radius 4px
- .badge — 8px, letter-spacing 1px, uppercase, 2px 6px padding, border-radius 2px
- .sidebar-item — 10px, letter-spacing 1.5px, uppercase, border-left 3px active
- .data-table — always table-layout:fixed, 10px cells, teal-500 headers, border-bottom subtle

## Layout
- App shell: .app (flex column, 100vh) > .topbar (48px) + .app-body (flex row)
- .app-body: .sidebar (168px fixed) + .main-content (flex:1, padding 20px)
- Content grids: .grid-2, .grid-3, .grid-4 using minmax(0, 1fr)
- Stack vertically: .stack (flex column, gap 14px)

## Do NOT
- Use border-radius > 4px on any element except badges (2px)
- Use any font other than Bebas Neue (display) or IBM Plex Mono (body)
- Hardcode hex colors — always reference CSS variables
- Use drop shadows, gradients, or blur effects
- Add rounded corners to single-sided borders
- Let table cells overflow — use text-overflow: ellipsis and fixed table-layout
